My Buddy Erik

Introduction :

My buddy Erik wants to play Minecraft so I set up a server for us to play on. I've committed my configuration to GitHub because it's so convenient! Can you make sure that everything is secure? https://github.com/VikeSec/vikeCTF-2024-minecraft-server

Solution :

Honestly I have no idea what to do at first, then I copied the given docker file and try to run it on my Kali

FROM itzg/minecraft-server
ENV EULA=true
EXPOSE 25565/tcp 25575/tcp
ENV ENABLE_WHITELIST=true
ENV WHITELIST=notch,ViktorViking
ENV TYPE=PURPUR
ENV USE_SIMD_FLAGS=true
ENV SNOOPER_ENABLED=false
VOLUME [ "/data" ]

I tried running the docker file and spawn a shell too to see if there's anything on it. But after going through most of the directory there is absolute nothing

Then I try looking at the github account once more to find any clues and then I stumbled upon something

On the commit history section shows that something was reverted. So I googled and found a tool called github-secrets. Run it and click the first link to reveal the flag
